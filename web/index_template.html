What Orbit2 does that Salesforce PRM / Impartner don't

1. Partner-side self-scorecard. Salesforce PRM and Impartner are built for vendors to manage their partners top-down. Orbit2 is built for the partner (Communardo) to score itself against what the vendor expects — nobody else builds this direction.
2. One transparent composite score. Vendor PRM tools show raw metrics; none blend them into a single weighted, fully-traceable performance number.
3. External sentiment monitoring built in. Press/market mentions of Communardo, auto-classified positive/negative, feeding directly into the visibility score — normally a separate media-monitoring tool.
4. Evidence Library with dedupe. Screenshots, exports, and docs are the source of truth, and re-uploading a newer file for the same metric/quarter automatically supersedes the old one — no vendor PRM tool tracks provenance like this.
5. Multi-vendor side-by-side view. Atlassian, AWS, Google, GSIs in one scorecard — vendor tools are single-tenant by design.
